Bem-vindo!

Este é um template básico de site com HTML, CSS e JavaScript organizados em arquivos separados.

Você pode usar esta estrutura como ponto de partida para criar páginas pessoais, portfólios, projetos escolares ou sites institucionais simples.

Sobre

Esta estrutura serve como base para desenvolvimento de sites. Edite os arquivos index.html, style.css e script.js para personalizar o projeto.

O objetivo deste template é manter o código organizado, separando a estrutura da página, o estilo visual e as funcionalidades interativas.
